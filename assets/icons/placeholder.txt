This directory contains icons and graphics for the website.

You can add:
- SVG icons
- PNG icons
- Favicon files
- Logo files

Recommended files:
- favicon.ico (16x16, 32x32, 48x48)
- favicon.png (32x32)
- apple-touch-icon.png (180x180)
- logo.svg (vector logo)
- logo.png (raster logo)

For better performance, use SVG icons when possible.
